Proyecto final del tercer corte de la materia "{}\+Aquitectura Computacional"{} \hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Problema principal}{Problema principal}}\label{md__r_e_a_d_m_e_autotoc_md1}
Se requiere un sistema eficiente, asequible y fácil de usar capaz de monitorear en tiempo real las condiciones ambientales, incluyendo temperatura, humedad, luz y la presencia de gases peligrosos. El sistema debe alertar a los usuarios cuando estas condiciones superen los umbrales preestablecidos. \hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Objetivos}{Objetivos}}\label{md__r_e_a_d_m_e_autotoc_md2}
\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{\texorpdfstring{Desarrollo del Hardware\+:}{Desarrollo del Hardware\+:}}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyItemize}
\item Utilizar un Arduino Mega como microcontrolador principal.
\item Integrar sensores de temperatura y humedad (DHT11), luz (fotoresistor) y gas.
\item Incorporar una pantalla LCD para la visualización de datos.
\item Añadir un potenciómetro y un teclado (keypad) para la configuración del sistema.
\item Implementar un LED RGB y un buzzer para las alertas sonoras y visuales.
\item Utilizar una protoboard y cables para implementar los circuitos necesarios. 
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysubsection{\texorpdfstring{Desarrollo del software\+:}{Desarrollo del software\+:}}\label{md__r_e_a_d_m_e_autotoc_md4}

\begin{DoxyItemize}
\item Programar la lógica de lectura y monitoreo de los sensores.
\item Desarrollar una interfaz de seguridad, menú principal y menú secundario para el usuario en la pantalla LCD.
\item Programar funciones para cada una de las implementaciones dentro del proyecto.
\item Implementar un sistema de alarma cada vez que se superan los umbrales establecidos para temperatura, gas y bloqueo del sistema.
\item Añadir medidas de seguridad, como bloqueo del sistema cuando se supere la cantidad de intentos al ingresar una contraseña o un limite de tiempo para ingresar la contraseña. 
\end{DoxyItemize}
